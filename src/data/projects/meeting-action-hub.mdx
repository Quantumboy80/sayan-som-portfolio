# Meeting Action Hub: Smart Meeting Synchronization

## Overview

Meeting Action Hub is an AI-powered helper tool designed to parse meeting transcriptions, automatically extract action items, responsibilities, and key takeaways, and instantly synchronize them to your personal or team workspace in Notion using Model Context Protocol (MCP) servers.

## What it does

- **Automated Extraction:** Paste your raw meeting transcript and let the AI extract exact actionable tasks.
- **Notion Integration:** Automatically formats tasks and pushes them directly to your Notion workspace databases.
- **MCP Server Support:** Native connection with Model Context Protocol servers for secure and standardized tool communication.
- **Clean Interface:** Designed with a vibrant, modern pastel/glassmorphic look for distraction-free task management.

## Why I built this

Meeting transcripts are often long and hard to navigate. Manually identifying who is doing what, setting deadlines, and migrating those tasks to productivity tools like Notion takes time and is prone to errors. Meeting Action Hub makes this process instantaneous.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Integration:** Notion API, MCP Server
